Form Update Data Warga

Halaman ini digunakan untuk membantu warga RT 06 / RW 13 Relife Greenville Cileungsi Cluster Lakesville dalam mengajukan pembaruan data keluarga secara tertib, aman, dan terdokumentasi.

Setiap pengajuan update data akan diverifikasi terlebih dahulu oleh pengurus RT sebelum data diperbarui pada database warga.

Isi Form Update Data

Jenis Data yang Dapat Diperbarui

Data Kepala Keluarga

Perubahan nama, nomor HP, pekerjaan, atau informasi kepala keluarga.

Data Anggota Keluarga

Penambahan, pengurangan, atau perubahan data anggota keluarga.

Data Domisili

Perubahan status tinggal, warga tetap, kontrak, pindah, atau domisili sementara.

Data Kontak

Perubahan nomor WhatsApp, nomor darurat keluarga, atau kontak yang dapat dihubungi.

Data Lainnya

Perubahan informasi lain yang dianggap perlu untuk administrasi lingkungan RT.

Persyaratan Umum

  • Pemohon merupakan warga RT 06 / RW 13 Relife Greenville Cileungsi Cluster Lakesville.
  • Mengisi form update data dengan informasi yang benar dan lengkap.
  • Mencantumkan nomor WhatsApp aktif untuk proses konfirmasi.
  • Menjelaskan data lama dan data baru yang ingin diperbarui.
  • Melampirkan dokumen pendukung apabila diperlukan, seperti KTP, KK, atau dokumen lain yang relevan.
  • Menunggu proses verifikasi dari pengurus RT sebelum data diperbarui.

Catatan: Data pribadi warga tidak ditampilkan secara terbuka di website. Pengajuan update data hanya digunakan untuk kepentingan administrasi lingkungan dan dikelola oleh pengurus RT yang berwenang.

Contoh Pengisian Update Data

Perubahan Nomor HP

Nomor lama: 0812-xxxx-xxxx. Nomor baru: 0813-xxxx-xxxx.

Penambahan Anggota Keluarga

Menambahkan anak/anggota keluarga baru beserta nama, tanggal lahir, dan hubungan keluarga.

Warga Pindah

Menginformasikan bahwa keluarga sudah pindah domisili agar data lingkungan tetap akurat.

Alur Update Data Warga

1. Isi Form Update Data

Warga mengisi form update data melalui tombol yang tersedia di halaman ini.

2. Data Masuk ke Pengurus

Data yang dikirim akan masuk ke rekap pengurus RT untuk diperiksa.

3. Verifikasi Data

Pengurus RT memeriksa kesesuaian data dan dapat menghubungi warga melalui WhatsApp jika diperlukan.

4. Data Diperbarui

Jika data sudah sesuai, pengurus akan memperbarui database warga.

5. Konfirmasi Selesai

Pengurus dapat memberikan konfirmasi kepada warga bahwa data telah diperbarui.

Ajukan Update Data

Silakan klik tombol di bawah ini untuk mengajukan pembaruan data warga. Pastikan data yang diisi benar dan lengkap agar proses verifikasi dapat berjalan lebih cepat.
